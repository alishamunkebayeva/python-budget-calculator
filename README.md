# Savings Goal Calculator (Python)

This Python program calculates how long it will take to reach a savings goal.  
The user enters their savings goal amount and the amount they plan to save each month.  
The program then calculates how many months are needed to reach the goal.

## Features
- Takes user input for savings goal and monthly savings
- Calculates the number of months needed to reach the goal
- Displays the result clearly to the user

## Sample Run

Enter your savings goal: 1000  
Enter the amount you will save each month: 200  

You will reach your savings goal in 5 months.
